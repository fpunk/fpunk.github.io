Perception

Jävlar Anamma (mental stress)
för att få skiten gjord, visa min övertygelse, ta mig i kragen, uppbåda mina yttersta krafter.
Förnuft
för att tänka klart och logiskt, sätta saker i perspektiv, felsöka, lösa gåtor.
Bildning
för att begripa hur det fungerar, vad det betyder, vad andra sagt om saken.
Charm
för att väcka sympati för mig och min sak, göra mig intressant och attraktiv, övertyga och manipulera.
Fusk
för att ljuga, smita, slarva, spela fult.
Kyla (mental stress)
för att övervinna mina känslor, upprätthålla min fasad, hålla linjen.
Överlevnad
för att hitta föda, lägga förband, skydda mig mot elementen, reparera min utrustning.

Styrka
för att lyfta skrot, sparka in dörrar, flexa, hålla greppet, hoppa långt.
Tålighet (fysisk stress)
för att ta stryk, motstå smärta, springa långt, dricka sprit, hålla andan, inte somna.
Smidighet
för att ducka, hålla balansen, röra mig i svår terräng, hoppa med precision.
Fingerfärdighet

Våld
för att orsaka eller försvara mig från fysisk skada.

Skjutjärn

Fartyg

Avgrundsfartyg

Hårdvara

Mjukvara

Pengar
